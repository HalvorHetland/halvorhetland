In this project the goal was to test for difference between the different scrolling directions on a computer mouse. We then created a test where the user would complete different tasks and with a timer. The first time with a natural scrolling direction on the mouse and the next time with a inverted scrolling direction. The conclusion was insignificant difference between the two directions.
